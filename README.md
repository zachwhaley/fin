Short for Griffin and pronounced Finn

fin is a simply made unix shell written in C

## What You Need

[GNU GCC](http://gcc.gnu.org/)

```bash
$ sudo dnf install gcc
```

## What it Does

Runs system commands from the shell.

Use Ctrl-C to exit the shell.

## How to Use it

```bash
$ make
```
